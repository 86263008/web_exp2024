服务端开发环境搭建

1 实验类型

验证型，2学时，必选实验

2 实验目的

掌握JSP运行环境的安装与配置；掌握STS4中Tomcat注册过程；掌握JSP页面的部署及调试过程

3 实验要求

完成各实验步骤并对关键结果截图

4 实验环境

Windows 7，JDK 8u311，Tomcat 9.0.54，STS4，Microsoft Edge/Chrome/Firefox等浏览器

5 实验步骤

安装JDK

为服务器安装Java运行环境，JDK使用比较成熟的版本（https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html），步骤如下：

  1. 按默认配置安装jdk-8u311-windows-x64.exe

  2. 为Java增加环境变量；在开始/设置/控制面板/系统/高级/环境变量/系统变量中新建两个环境变量：

    JAVA_HOME:{JAVA_HOME}

    JRE_HOME:{JAVA_HOME}\jre

  3. 将JAVA_HOME添加到Path变量

  4. 确认Java安装成功，进入cmd命令行运行如下命令查看Java版本号：

java –version

安装Tomcat

安装Tomcat服务器。在官网（https://tomcat.apache.org/download-90.cgi）下载安装包。Windows下有两种方式：安装包、压缩包，本实验选择压缩包，步骤如下：

  1. 安装（解压）apache-tomcat-9.0.71-windows-x64.zip到目录C:/App/Tomcat 9.0

    注意端口号为:8080

  2. 在控制台（推荐Vscode终端）运行bin/startup.bat启动Tomcat

  3. 打开浏览器访问http://localhost:8080，确认Tomcat首页是否能正常显示

安装STS 4

安装集成开发环境STS4，在官网（https://spring.io/tools）下载适合本地电脑的版本，以Spring Tools 4 for Eclipse Windows X86_64为例，具体步骤如下：

  1. 安装（解压）sts-4.17.2.RELEASE.zip到目录C:/App/Sts4

  2. 启动SpringToolSuite4.exe，确认软件能正常运行

  3. 打开菜单Help/Install New Software...，在Work with:输入Juno - http://download.eclipse.org/releases/juno，在下方组件中选择Web, XML, Java EE and OSGi Enterprise Development，并在下一级组件中勾选以下组件：

    1. Eclipse Java EE Developer Tools
    2. Eclipse Java Web Developer Tools
    3. Eclipse Web Developer Tools
    4. Eclipse XML Editors and Tools

创建JSP测试工程

创建一个JSP工程，在页面中使用Java代码输出问候字符串，步骤如下：

  1. 创建工作目录：C:/{学号}

    后续文件均存放在工作目录

  2. 启动STS4（Workspace指向工作目录），创建名为HelloJsp的Web工程（File/New/Project/Web/Dynamic Web），向导过程中修改配置如下：

    选项 值
    Project Name HelloJsp
    Generate web.xml deployment descriptor 选中

  3. 注册Tocmat服务器；在Servers视图（默认在STS4界面下方，也可通过菜单Window/Show View/Servers打开）添加Apache/Tomcat 9

    关键设置：1. 指定Tomcat 9安装目录；2. JRE选择jre

  4. 配置服务器运行时，在工程的快捷菜单中选择Build Path/Configure Build Path.../Libraries/Class Path/Add Library/Server Runtime/Apache Tomcat 9.0

  5. 创建webapp/index.jsp，参考代码如下：
