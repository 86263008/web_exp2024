JSP的隐含对象实验

1 实验类型

验证型，2学时，必选实验

2 实验目的

掌握JSP基本语法；掌握对JSP各隐含对象的功能；掌握JSP隐含对象的使用方法；理解前后端数据传输过程

3 实验要求

编写代码测试request对象的功能；编写代码测试response对象的功能；编写代码测试session对象的功能；编写代码测试application对象的功能；编写代码测试out对象的功能

4 实验环境

Windows 7，JDK 8u311，Tomcat 9.0.54，STS4，Visual Studio Code，REST Client, Microsoft Edge/Chrome/Firefox等浏览器

5 实验步骤

REST Client安装

打开Visual Studio Code，安装扩展REST Client

创建测试工程

为实验创建名为BuiltInTest的测试工程，步骤如下：

  1. 创建工作目录：C:/{学号}

    后续文件均存放在工作目录

  2. 在STS4中创建的Dynamic Web工程：BuiltInTest

    注意：勾选Generate web.xml deployment descriptor
